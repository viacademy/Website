    • COURSE DESCRIPTION

    MySQL is an open source database that is widely used in many applications and it is completely free. In our MySQL Training we are covering from the basic SQL Queries and we are integrating that into PHP. PHP, MySQL with Apache Server is the perfect combination for any web applications.

    Our course provides using and maintaining MySQL database server as well as backing up, recovery, configuration, and improvement. We are giving MySQL training with class room as well as online sessions. We have convenient batches with versatile batch timings.



    MYSQL COURSE SYLLABUS

    MYSQL

    • Database concepts
    • Working with PHPmyAdmin
    • Mysql Data types
    • Creating Database and table
    • Connecting mysql with php
    • Queries – Insert, select, update and delete
